# L20 Decision Demo

Этот пример показывает, как L20 (Resonant Decision Orchestrator) выбирает
следующее действие на основе:

- внутреннего состояния (время, смысл, фаза),
- внешнего давления (L19),
- оси удачи / выравнивания потока (L21),
- списка возможных вариантов.

Пример сценария:

- организм вдохновлён, но физически устал,
- внешнее давление средней силы и относится к "challenge",
- окно удачи открыто, но фаза неустойчивая,
- есть три варианта действий: `deep_coding_push`, `family_evening`, `money_focus`.

Ожидание:

- L20 выберет `family_evening` в режиме `stabilize` с высоким `resonance_score`,
  так как это действие повышает стабильность, сохраняет смысл и лучше согласовано
  с текущей осью удачи.
