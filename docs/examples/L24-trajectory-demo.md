# L24 demo — гармонизация траекторий на живых данных

Этот пример запускает **Trajectory Harmonizer Layer (THL)** на трёх кандидатах и снимке среды L23. Его можно выполнить через `tsx` / `ts-node`:

```bash
npx tsx docs/examples/L24-trajectory-demo.ts
```

## Данные демо

- **Снимок среды (L23)**: финансы в метастабильной фазе, семья в fluid, здоровье заморожено.
- **Вектор воли/удачи (L21)**: фокус на стабильности и долгосрочных решениях, умеренный риск, желание мягких изменений.
- **Траектории**:
  - **T1 — Aggressive financial pivot**: высокая потенциальная выгода, но опирается на finance+health → высокий риск.
  - **T2 — Gentle course correction**: мягкая корректировка через finance+family, совпадает с тегами воли.
  - **T3 — Maintain current state**: минимальный риск и выгода, но не поддерживает drive к изменениям.

## Ожидаемое поведение

- T1 получает высокий риск из-за frozen health и общей турбулентности.
- T3 штрафуется за низкий gain и слабое совпадение с `changeDrive`.
- T2 балансирует резонанс с fluid/metastable регионами и совпадает с фокус-тегами, поэтому выходит победителем.

Запустив пример, вы увидите выбранную траекторию (T2) и таблицу метрик по всем кандидатам.
