# L26 — Organelle Mesh / Cytoskeleton Layer

L26 моделирует органеллы клетки и цитоскелет LIMINAL-организма. Если L25 (CellKernel) решает, **что** делать и сколько энергии выделить, то L26 отвечает за то, **какие органеллы возьмут работу**, как распределить нагрузку и сохранить форму организма.

## Зачем нужен слой L26
- держать реестр органелл и их профили (ёмкость, энергия, здоровье);
- принимать задачи от L25/DecisionBus и сопоставлять их с подходящими органеллами;
- защищать организм от перегрузок, разносить нагрузку по профилям;
- хранить "форму" организма через простой граф связей (цитоскелет);
- строить метрики здоровья и баланса органелл.

## Био-аналогия → инженерные роли

| Органелла (био)    | Роль в L-EDGE                                            |
|--------------------|----------------------------------------------------------|
| Цитоскелет         | Граф связей между модулями, форма организма              |
| Митохондрии        | Энергетические модули (CPU/GPU пулы, воркеры, кэши)      |
| Рибосомы           | Исполнение задач (job-queues, pipeline-стадии)           |
| Аппарат Гольджи    | Маршрутизация/упаковка результатов, события, webhooks    |
| ЭПР                | Транспорт данных (шины, event-bus, data pipelines)       |
| Лизосомы           | Очистка, отмена, retries, dead-letter                    |
| Вакуоли/резервуары | Буферы ожидания, rate-limit, backpressure                |

## Как L26 работает вместе с L21–L27
- **L24 → L25 → L26**: L24 выбирает сценарий, L25 оценивает энергетику и собирает `CellActionPlan`, L26 превращает его в `OrganelleTask[]` и раздаёт органеллам.
- **L27**: обновляет лимиты энергии/здоровья органелл (витамины/минералы), может временно бустить нужные органеллы.
- **L21–L23**: используют метрики L26 (баланс, перегрузка) для корректировок потока, вязкости информации и внутреннего нарратива.

## Метрики L26
- **avgHealth** — среднее здоровье органелл (0..1).
- **loadBalanceIndex** — равномерность нагрузки (1 — идеально ровно).
- **congestionScore** — насколько сетка забита (0 — свободно, 1 — перегруз).
