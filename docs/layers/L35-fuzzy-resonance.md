# L35 Fuzzy Resonance Layer

**Зачем:** L35-Tantra уже строит ткань связей и считает численные метрики (flow/resilience/insight/luck). Fuzzy Resonance переводит
эти числа в «язык ощущений»: удачные коридоры, стабилизирующие косы, хаотичные мосты и латентные нити инсайта.

**Как работает:**
- `toFuzzy` разбивает диапазон 0..1 на low/medium/high.
- `profileLink` превращает `TantraLink` + `OrganismTone` в мягкий профиль с тегом (`lucky_corridor`, `stability_braid`,
  `chaotic_bridge`, `deep_insight_thread`, `latent_potential`, `shallow_noise`) и коротким комментарием.
- `profileFabric` собирает доминирующие теги по ткани, усредняет удачу (`overallLuck`) и даёт готовую карту для UI/агента.

**Использование:**
- UI: подсветить связи по тегам (зелёный коридор, синий стабилизатор, янтарный хаос-мост).
- Агент: выбирать шаги не только по силе, но по `lucky_corridor + stability_braid`, избегать `shallow_noise`.
- Человек: видеть карту удачи/стабильности/хаоса дня или спринта.

**Философия:** нити и резонансы → теория струн; нечёткая логика вместо бинарности; удача как плотность резонансных пересечений;
система чувствует, какие связи живые, какие — шум.
