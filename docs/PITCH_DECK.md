# LIMINAL EDGE — Pitch / One-pager

## 0. One-pager (инвесторы)
- **Что:** живой edge-слой интернета. Узлы — клетки, маршрутизация — смысловые паттерны, атаки → иммунитет.
- **Почему сейчас:** всплески AI/IoT трафика делают статические сети уязвимыми. Нужен организм, не машина.
- **Как:** SomaSeed клетки + FlowBalancer + Guard + GraphDNS + CDN Lite + Worker Cells. Все учится на паттернах.
- **Рынок:** CDN/WAF/Edge ~$40B; замена SPOF-архитектур, экономика ресурсоэффективности.
- **Тракшн (план):** MVP 2 недели, первые пилоты — gaming/fintech/e-commerce, SLA-бандлы на M6.
- **Моат:** самообучение на паттернах, резонансная защита, мигрирующие воркеры, графовое DNS.

## 1. Визия
- Internet Layer 3: адаптивный живой слой, растущий и самозалечивающийся.
- От SPOF к «рою» клеток. От правил к паттернам. От защиты к иммунитету.

## 2. Проблема
- Централизованные CDN и WAF — точка отказа и ручные конфиги.
- Пики трафика и DDoS ломают статическую архитектуру.
- Высокая стоимость over-provisioning и поддержания SLA.

## 3. Решение
- **SomaSeed**: клетки-агенты, делятся/сворачиваются по спросу.
- **FlowBalancer**: маршрутизация по смыслу (паттерны), предсказывает пики.
- **Liminal Guard**: резонансный WAF, fuzzy-анализ аномалий.
- **GraphDNS**: взвешенные ответы, мгновенный failover.
- **CDN Lite + Worker Cells**: кеш и код рядом с трафиком, миграция воркеров.

## 4. Продукт (MVP → M6)
- 2 недели: дышащий сервер (Node/Rust + InfluxDB + Pythia + лёгкий CDN).
- 6 месяцев: самозалечивание, обучаемый CDN, мигрирующие воркеры, enterprise SLA.

## 5. Технологический снэпшот
- **Data plane:** Rust-агенты + Node gateway; локальные кеши; State/Pattern buses.
- **Control plane:** Pythia-анализ паттернов; GraphDNS; оркестрация роста/сворачивания.
- **Observability:** метрики в InfluxDB, аномалии LG, дашборд карты узлов.

## 6. Go-to-market
- Пилоты: high-RPS продукты (гейминг, финтех, e-commerce flash sales).
- Ценность: стабильные задержки в пиках, автоматическое самозалечивание, снижение расходов на over-provisioning.
- Модель: usage + SLA-бандлы; опции частного кластера.

## 7. Команда/статус
- Сейчас: основатель + первые инженеры.
- Ближайшие 2 недели: MVP, демо иммунитета к всплескам трафика.
- 6 месяцев: 5–10 пилотов, первые enterprise контракты.
