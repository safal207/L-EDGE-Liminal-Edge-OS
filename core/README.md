# Core Layer

Базовые типы, события и константы, которые используются всеми слоями LIMINAL EDGE. Здесь определяются ID узлов и сервисов,
корреляционные идентификаторы, форматы Edge/Runtime/Resonance событий и контракты сообщений LTP. Другие слои импортируют
эти типы для консистентности и трассируемости.
