# Interface Layer

Backend для Liminal Resonance Interface (LRI). В итерации 1 реализован Express API с эндпоинтами `/api/edge/events` и
`/api/system/health`. Здесь же появятся WebSocket/SSE для real-time обновлений панели оператора. Визуальный UI находится в
https://github.com/safal207/LRI-Liminal-Resonance-Interface.
